FRIEREN: A Lightweight System for Face Resizing Image Detail Quality Evaluation via Robust Estimation of Image Naturalness

IEEE APCCAS 2025
Yuan-Kang Lee * , Kuan-Lin Chen * , Jian-Jiun Ding
Graduate Institute of Communication Engineering, National Taiwan University
*Indicates Equal Contribution

FRIEREN (Face Resizing Image Detail Quality Evaluation via Robust Estimation of Image Naturalness), a novel IQA for detail quality evaluation, effectively and accurately evaluates the face image detail quality scaled by different interpolations with low computational complexity.

Abstract

FRIEREN (Face Resizing Image Detail Quality Evaluation via Robust Estimation of Image Naturalness) is a novel image quality assessment (IQA) system specifically designed for evaluating detail quality in face images. This lightweight system effectively and accurately assesses face image detail quality when scaled using different interpolation methods, while maintaining low computational complexity.

Traditional IQA metrics often fail to capture the perceptual quality of face images after resizing operations. FRIEREN addresses this gap by focusing on robust estimation of image naturalness, providing a more accurate evaluation framework for face image processing pipelines.

Method

System Architecture

FRIEREN employs a robust estimation framework that analyzes multiple aspects of image naturalness:

  1. Feature Extraction: The system extracts key features from face regions that are sensitive to interpolation artifacts
  2. Naturalness Estimation: Uses machine learning models trained on human perception data to estimate image naturalness
  3. Detail Quality Scoring: Combines multiple quality indicators to produce a comprehensive detail quality score

Technical Approach

  • Lightweight Design: Optimized for real-time processing with minimal computational overhead
  • Interpolation Robustness: Tested across various interpolation methods (nearest neighbor, bilinear, bicubic, Lanczos)
  • Perceptual Alignment: Strongly correlated with human subjective quality assessments

Results

Performance Evaluation

FRIEREN demonstrates superior performance compared to existing IQA methods:

  • Accuracy: High correlation with human subjective scores (PLCC > 0.90)
  • Efficiency: Processing time reduced by 60% compared to state-of-the-art methods
  • Robustness: Consistent performance across different interpolation algorithms

Key Findings

  1. The system accurately identifies quality degradation in resized face images
  2. Low computational complexity makes it suitable for real-time applications
  3. Robust across various scaling factors and interpolation methods
